Actividad 8: 💌 El Mensaje Secreto de Hera 💌

Hola, jóvenes criptógrafos! 🕵️‍♂️ Hoy vamos a ayudar a Hera, la reina de los dioses, a enviar un mensaje secreto. ¿Están listos para combinar letras y números en un enigma digno del Olimpo?

Hera quiere enviar un mensaje que solo los dioses puedan entender, mezclando letras y números. Usaremos nuestras habilidades en Python para crear este mensaje cifrado.

Hera:

📜✨ Hola, astutos programadores. Soy Hera, y necesito vuestra ayuda para enviar un mensaje secreto. Combinad letras y números de manera que solo los más sabios puedan descifrarlo. 📜✨

Pistas:

Para esta tarea, recordad cómo concatenar strings y números. Por ejemplo, si queremos combinar el número 12 con la palabra 'Olimpo', podemos escribir `str(12) + 'Olimpo'`.

Instrucciones:

  1. Configura tu Espacio: Abre tu editor de Python y crea un nuevo archivo llamado hera_mensaje.py.
  2. Declara las Variables: Define variables para almacenar partes del mensaje, incluyendo tanto strings como números. 
    parte1 = "El secreto es "
numero = 7
parte2 = " al amanecer"
  3. Concatenar Mensaje: Combina las partes para formar el mensaje secreto. 
    mensaje_secreto = parte1 + str(numero) + parte2
  4. Imprimir Resultado: Muestra el mensaje secreto usando la función print(). 
    print(mensaje_secreto)
